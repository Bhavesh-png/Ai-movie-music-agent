# 🤖 AI Multi-Feature Assistant (n8n)

## 🚀 Overview

This project is an AI-powered assistant built using n8n that can provide movie recommendations, music suggestions, latest news, and weather updates through a conversational interface. It demonstrates how AI agents can interact with multiple APIs inside an automation workflow.

## ✨ Features

* 🎬 Movie recommendations (Bollywood & Hollywood)
* 🎵 Music suggestions (songs, artists)
* 📰 Latest news updates (Tech, World, etc.)
* 🌦️ Weather information (city-based)
* 🤖 AI-powered chat interface

## 🛠️ Tech Stack

* n8n (Workflow Automation)
* HTTP APIs (OMDb, iTunes, News API, Weather API)
* AI Agent (Tool-based interaction)

## 📦 Setup

1. Install n8n:

```id="v3l6s2"
npm install -g n8n
```

2. Start n8n:

```id="q9g1kf"
n8n
```

3. Import workflow:

* Open n8n UI
* Import `workflow.json`

4. Add API keys:

* OMDb API key
* News API key (if used)
* Weather API key (if used)

## 💬 Usage

You can ask:

* "Suggest Bollywood movies"
* "Recommend songs"
* "Latest tech news"
* "Weather in Mumbai"

## 🎯 Purpose

This project showcases how AI agents can be combined with automation tools like n8n to build powerful, real-world assistants without complex UI development.
